# Tauri + Vue 3 + TypeScript + Pinia 基础开发模板

一个基础的tauri开发模板，具备国际化多语言切换，暗黑和亮白主题切换，注册登陆页面美化，动态路由自动构建左侧菜单列表，
Pinia状态管理等等功能，开箱即用

## 使用方式
代码下载后，使用pnpm 安装依赖
```
pnpm install
```

运行程序：
```
pnpm develop
```
